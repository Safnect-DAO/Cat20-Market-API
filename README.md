# Cat20-Market-API

Cat20-Market-API 用于构建Market模块的工具包，提供了多条件（根据地址、TokenId、订单类型）查询多类型（Open、Cancel、Deal）的Order、Make Order、Take Order、Cancel Order、和交易数据统计功能。

订单的生命周期：Open (Make order) - Deal (Take Order) | Canceled (Cancel order)
